Industrial Ethernet: Half Duplex vs Full Duplex
Project Overview

This project presents a practical Industrial Ethernet solution to address communication challenges in a manufacturing environment. The objective is to analyze and compare Half Duplex and Full Duplex communication and recommend the most suitable approach for reliable industrial automation.

The project demonstrates network design, configuration, packet analysis, and host discovery using Cisco Packet Tracer, Wireshark, and Nmap.
Scalability

The proposed Full Duplex switched network offers:

Easy expansion
Better bandwidth utilization
Support for Industrial IoT devices
Reliable communication for larger industrial environments
Security

The proposed solution supports:

Managed Switching
Port Security
VLAN Segmentation
MAC Address Learning
Network Monitoring
Results

The network was successfully designed and configured.
The Cisco CLI verified both Half Duplex and Full Duplex configurations.
Wireshark successfully captured Ethernet communication.
Nmap successfully discovered active devices on the network.
Based on performance, scalability, and reliability analysis, Full Duplex is the preferred communication mode for Industrial Ethernet applications.
